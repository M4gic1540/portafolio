# Portafolio Personal — Tomás González Borje

Portfolio profesional desarrollado con React, TypeScript y Tailwind CSS v4. Presenta mi experiencia, habilidades técnicas y proyectos realizados.

## 🚀 Stack Tecnológico

- **React 19** — Biblioteca UI con componentes funcionales
- **TypeScript 5.9** — Tipado estático
- **Tailwind CSS v4** — Estilos utilitarios con tema personalizado
- **Vite 7** — Build tool y servidor de desarrollo

## 📁 Estructura del Proyecto

```
src/
├── assets/              # Imágenes y recursos estáticos
├── components/
│   ├── Navbar.tsx       # Navegación fija con menú mobile
│   ├── Hero.tsx         # Sección principal con foto de perfil
│   ├── Experience.tsx   # Experiencia profesional y educación
│   ├── Skills.tsx       # Habilidades técnicas y certificaciones
│   ├── Projects.tsx     # Proyectos personales y empresariales
│   ├── Contact.tsx      # Información de contacto
│   └── Footer.tsx       # Pie de página con navegación y redes
├── App.tsx
├── main.tsx
└── index.css            # Tokens de diseño y configuración Tailwind
```

## ⚙️ Instalación y Uso

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev

# Build para producción
pnpm build

# Preview del build
pnpm preview
```

## ✨ Características

- Diseño responsive optimizado para móvil, tablet y desktop
- Tema oscuro con paleta de colores personalizada (índigo/violeta)
- Animaciones suaves en anillos del avatar
- Optimización LCP con preload dinámico de imagen
- Accesibilidad con atributos `aria-label` y semántica HTML
